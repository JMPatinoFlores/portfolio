# Personal Portfolio — Jessica Patiño

A modern, interactive frontend developer portfolio built with Next.js, TypeScript, and Tailwind CSS.

Designed to showcase my experience building production-ready applications, including ERP/CRM systems, booking platforms, and full-stack dashboards. The portfolio features custom illustrations, animated UI elements, and a cohesive visual identity inspired by late-night development.

## Features

- Responsive design optimized for all devices
- Interactive project showcase with live demos and source code
- Animated UI elements and micro-interactions
- Custom illustration system and visual branding
- Component-based architecture

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- shadcn/ui
